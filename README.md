 # Optimization Techniques Project
## Overview
This project demonstrates and compares several optimization algorithms used in numerical optimization.
## Implemented Methods
- Gradient Descent (GD)
- Steepest Descent with Exact Line Search
- Newton's Method
- Quasi-Newton Method (BFGS)
## Technologies Used
- Python
- NumPy
- SciPy
- Matplotlib
- Jupyter Notebook
## Objectives
- Understand optimization techniques.
- Compare convergence behavior of different algorithms.
- Visualize optimization paths and results.
